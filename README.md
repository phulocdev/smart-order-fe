## Smart Order - Client

### Mô tả
Frontend ứng dụng quản lý nhà hàng sử dụng Next.js + Tailwind CSS + React Query.

### Cấu hình môi trường
Tạo file `.env` với nội dung:

```
NEXT_PUBLIC_BACKEND_URL=http://localhost:8000/api/v1
NEXT_PUBLIC_NEXT_API_URL=http://localhost:3000
NEXT_PUBLIC_WEBSOCKET_URL=http://localhost:8000
NEXT_PUBLIC_API_KEY=soaSpring2025
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
NEXTAUTH_SECRET=phamphuloc
NEXTAUTH_URL=http://localhost:3000
```

### Khởi động
```bash
npm install
npm run dev
```
